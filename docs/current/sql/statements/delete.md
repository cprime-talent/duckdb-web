# Delete Statement
DELETE FROM - remove rows from a table in the database

### Examples
```sql
-- remove the rows matching the condition "i=2" from the database
DELETE FROM tbl WHERE i=2;
-- delete all rows in the table "tbl"
DELETE FROM tbl;
```
